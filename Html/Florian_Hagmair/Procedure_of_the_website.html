Procedure of the website

Here will be explained how the game works and what procedure the main website has.

First there is an introduction at the top, what the idea of this website is and what we want to achieve with it.


History

The website 'History' writes about the history of all world cups before.

It is written about the start of the first world cup and in which country.

The question will also be clarified who whon the first world cup in the history and there is then a list of the following winners and top scorers of each world cup.

It is also written about the favourit teams and how many timey they have won the world cup.

At the end of the website it is written about the records set by various teams but also about the special players records.


Procedure of the world cup system

The website 'Procedure of the world cup system' writes about the system of a world cup.

There is explained how the group stage system works and how the points are distributed after a win, a lose or a draw.

It is also be written how long a match go and the question is be answerd how many teams are playing in the world cup und how many teams are in ech group.

Also it is written about the system of the knockout round because there are differnt rules as in the group stage.

The most interestin question what you get for the victory and how much money is it will be answered on the website below.

You can click on the link above to get more detailed informations about the questions here.


After all these pages the tree of the world cup is listed and there are some nice informations.


Tree of the world cup

The tree of the world cup is a overview with the whole groups and their teams.

This way you can see perfectly which country is with whom in a group.

A nice feature is that you can click on any team you want and then you will get the country's squad with all players.

Then the game begins. First you can see all game encounters and then you can submit your own tip for each game.
